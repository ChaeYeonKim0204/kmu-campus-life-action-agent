# 국민대학교 졸업 도우미

RAG 기반 개인 맞춤형 졸업 진단 서비스

## 빠른 시작

### 1. 의존성 설치
```bash
pip3 install -r requirements.txt
```

### 2. API 키 설정
```bash
cp .env.example .env
# .env 파일에 OPENAI_API_KEY 입력
```

### 3. 요람 PDF 준비
요람 PDF 원본은 용량이 커서 Git 저장소에 포함하지 않습니다.

국민대학교 공식 요람/규정집 페이지에서 2025 요람 PDF를 내려받아 아래 파일명으로 `test/` 폴더에 둔 뒤 인덱싱을 실행하세요.

```text
2025국민대학교요람_20250910.pdf
```

### 4. 요람 인덱싱 (최초 1회, 약 10~20분 소요)
```bash
python3 1_build_index.py
```
- `chroma_db/` 폴더와 `index_stats.json`이 생성됩니다.
- OpenAI API 비용: 약 $0.5~1.0 (text-embedding-3-small 기준)

### 5. 앱 실행
```bash
streamlit run 3_app.py
```

## 4대 에이전트

| 에이전트 | 기능 |
|---------|------|
| 졸업 진단 | 성적증명서 기반 졸업 가능 여부 판정 |
| 대체 이수 | 폐강/수강실패 과목의 대체 과목 탐색 |
| 마이크로 디그리 | 추가 취득 가능한 소학위 트랙 발굴 |
| 직무 역량 번역기 | 이수 과목 → 자기소개서 역량 변환 |

## 주의사항
- 모든 진단 결과는 **참고용**이며 최종 졸업 승인은 교학팀 확인 필요
- 성적증명서 PDF는 로컬에서만 처리되며 외부 전송 없음
